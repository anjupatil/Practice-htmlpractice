HTML Block & Inline Elements

  • Every HTML element has a default display value, depending on what type of element it is.

  • There are two display values:

    • Block
    • Inline

  • Block-level Elements

    • A block level element always starts on the new line.Browsers automatically add some space(a margin) before and after the element.
    • A block level element always takes up the full width available(stretches out to the left and right as far as it can).
    • Two commonly used block elements are:
      • p tag:defines paragraph.

        • Hello World, this is a paragraph.

      • div tag:defines a division or a section.
        • Hello World,this is a division.

  • Inline Elements

    • An inline element does not start on a new line.
    • An inline element only takes up as much width as necessary.
    • Note:An inline element cannot contain block level element.
    • This is a span element.

  • Div Element

    • The div element is often used as a container for other HTML elements
    • The div element has no required attributes, but style, class and id are common.

    • HyperText Markup Language or HTML

      HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
