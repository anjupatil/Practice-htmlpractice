HTML Images

  • HTML Images Syntax:

    • img does not have a closing tag

    • src: specifies path to the image.

    • alt: specifies an alternate text for the image.If a browser cannot find an image, it will display the value of the alt attribute.

    • The width and height attributes always define the width and height of the image in pixels.

    • HTML allows animated GIFs.
