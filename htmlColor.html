HTML Colors

  • HTML colors specified with pre-defined color names or with RGB,HEX,HSL,RGBA or HSLA values.

    HSL stands for hue, saturation, and lightness. HSLA color values are an extension of HSL with an Alpha channel (opacity).

  • Color names such as

    Bg color set to MediumSeaGreen.

  • background-color: Background color sets the background color for HTML elements.

    Background color of this paragraph is set to tomato color.

  • color: sets the color of text

    Text color is set to DodgerBlue.

  • border: Border color

    This paragraph's border is set to 2px, solid, violet values.
