CSS=Cascading Style Sheets

  • CSS control the layout of multiple web pages all at once.

  • Cascading means a Style applied to a parent element will also apply to all children elements within the parent. Example: color of body text=blue=>all headings ,paragraphs and other text elements also get the same color(unless you spexify something else).

Using CSS

CSS used in 3 ways:

  • Inline= by using the style attribute inside HTML Elements

  • Internal= by using the style element inside head section.

  • External= by using a link element to link to an external CSS file.

The most common way to add CSS is to keep the styles in External CSS files because it applies styles to multiple web pages all at once.
