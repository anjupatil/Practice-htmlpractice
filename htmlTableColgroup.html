HTML Table Colgroup

  • The colgroup element is used to style specific columns of a table.

  • HTML Colgroup

    • To style first two cols of a table,use colgroup and col elements.
    • The colgroup element used as a container for column soecifications.
    • Each group is specified with col element.
    • The span attribute specifies how many columns that get the style.
    • The style attribute specifies the style to give the columns.
    • Note: The colgroup element must be a child of a table element and should be placed before any other table elements, like thead,tr,td etc,but after the caption element,if present.
